# 1주차 과제 - 베이스볼 게임

## 요구사항

- 컴퓨터는 1~9 중 서로 다른 세 정수를 정답으로 선택한다.
- 플레이어는 정답을 예상해 세 자리 숫자 형태로 입력하고,
- 컴퓨터는 실제 정답과 대조하여 적절한 힌트(스트라이크, 볼, 낫싱)을 제공한다.
- 이 과정을 반복해, 컴퓨터가 선택한 3개의 숫자를 모두 정확히 맞히면 게임이 종료된다.
- 게임이 종료된 이후 다시 시작하거나, 프로그램을 완전히 종료할 수 있다.
