# 미션 - 숫자 야구 게임

## 기능 목록
> 야구 게임을 구현하는 데 필요한 책임(행동의 모음)을 기준으로 기능을 계획했습니다.
* **3자리 난수 생성**
  * 1부터 9중 랜덤으로 한자리의 수
  * 서로 다른 수로 구성되야 한다.


* **스트라이크, 볼, 낫싱 구현**
  * 같은 수 && 같은 자리 == 스트라이크
  * 같은 수 && 다른 자리 == 볼
  * 전혀 다른 수 == 낫싱
  * 전부 같은 수 == 승리 


* **플레이어 숫자 입력 및 예외 처리**
  * 입력
    * 게임 재시작 / 종료 입력 (시작(1) / 종료(2) 숫자 입력)  
    * 야구 숫자 입력 (플레이어 1부터 9까지 서로 다른 숫자 입력)
  * 예외처리 
    * 입력한 숫자가 서로 다른 숫자인지
    * 입력한 값이 숫자인지
    * 입력된 값의 길이가 3인지


* **게임 구현**
  * 야구 게임 한판(단위) 실행 구현
  * 야구 게임 전체적인 실행 구현
