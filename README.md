# 미션 - 숫자 야구 게임

## 기본 구현 사항

- 1 ~ 9 사이의 난수 3개 생성 (중복 X)
- 사용자 3자리 입력
  - 입력된 값이 숫자인지 확인 (문자 X)
  - 입력된 값의 길이가 3자리인지 확인
  - 입력된 값의 중복 확인 (추가)

- 볼, 스트라이크 확인
  - 볼 : 입력숫자 in 난수 and 입력위치 != 난수위치
  - 스트라이크 : 입력숫자 in 난수 and 입력위치 == 난수위치

- 힌트 및 결과 제공  
  - 3스트라이크일 경우 : 결과 제공 및 종료
  - 이외의 경우는 힌트 제공 및 반복
    - 스트라이크 혹은 볼이 존재 : 힌트 제공 ( O스트라이크 , O볼, O볼 O스트라이크 )
    - 스트라이크, 볼 존재 X : "낫싱"

- 게임 반복여부
  - 1 : 새게임
  - 2 : 종료
  - 1, 2에 대한 확인 필요
