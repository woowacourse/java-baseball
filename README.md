# 우아한 테크 코스 <br> 온라인 과제 1주차 Java Baseball

## 기능 목록  
### 1. 컴퓨터 난수 생성  
 	1. 크기가 3인 computerNumbers 배열 생성  
		* index 0: 100의 자리  
		* index 1: 10의 자리  
		* index 2: 1의 자리  
 	2. 세 변수를 난수로 초기화  
 	3. 0이 아니고 값이 서로 다르도록 난수 생성 반복  
 	
### 2. 사용자 숫자 입력
 	1. 크기가 3인 userNumbers 배열 생성
 	2. 사용자가 입력한 숫자, 배열에 삽입  
 		* index 0: 100의 자리
 		* index 1: 10의 자리
 		* index 2: 1의 자리
 		  
### 3. 사용자가 입력한 숫자 확인
 	1. 3자리 숫자인지 확인  
 	2. 각 자리가 서로 다른 지 확인  
 	3. 조건에 맞지 않다면 다시 입력 받는다  
 	
### 4. 볼/스트라이크 검사
 	1. 볼 검사  
 		- computerNumbers와 userNumbers의 index가 서로 다를 경우 확인  
 	2. 스트라이크 검사  
 		- computerNubmers와 userNumbers의 index가 서로 같을 경우 확인
 	3. 볼 검사, 스크라이크 검사 함수 호출
 	
### 5. 결과 출력
 	1. strike와 ball 개수 확인, 결과 출력
 	
### 6. 게임 진행
 	1. strike 개수가 3이 아니라면, 0 반환  
 	2. strike 개수가 3이라면 결과 출력, 게임 진행 여부 확인
