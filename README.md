숫자야구게임 기능 구현
==============


시작
----
- 게임에 사용될 랜덤한  1-9사이의 **숫자 3개** 생성
	+ 생성된 숫자 중 **중복된 것이 있으면** 다시 생성

입력
----
- **3자리 숫자**를 입력받음
  + 1-9 사이의 숫자로만 입력되었는지 검사
  + **숫자가 아닌 문자, 0, 3자리가 아니면** 다시 입력

과정
----
- 같은 자리에 같은 숫자가 있으면 `스트라이크`
- 다른 자리에 숫자가 있으면 `볼`
- 같은 숫자가 하나도 없으면 `낫싱`
- 정답을 맞추면 **게임 종료**

종료 및 재시작
--------
- 1을 입력하면 재시작, 2를 입력하면 종료
  + **그 외의 숫자 또는 문자를 입력**하면 다시 입력
