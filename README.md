# 기능 목록

1) 컴퓨터가 1 ~ 9 까지의 서로 다른 수로 이루어진 임의의 3자리 수를 선택하는 기능(feature/number-selection)

2) 플레이어에게 1 ~ 9까지의 서로 다른 수 3개를 입력받는 기능(feature/player-input)

3) 플레이어가 선택한 수가 컴퓨터가 선택한 수와 비교해 어느 정도 맞는지 판정하는 기능(feature/guess-judgement)

4) 판정 결과를 사용자에게 보여주는 기능(feature/result-display)

5) 게임의 종료 여부를 판별하는 기능(feature/termination-judgement)

6) 플레이어가 재시작과 종료를 선택하는 기능(feature/restart-selection)
