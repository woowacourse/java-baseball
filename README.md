## 🖊 기능 구현 목록

### 기능 목록

- 숫자 야구 게임 실행
- 중복되지 않는 1 ~ 9 사이의 임의의 수 3개 생성
- 사용자 숫자 입력
- 사용자 숫자와 난수 비교
    - 스트라이크, 볼 카운트
    - 하나도 없는 경우
    - 모두 맞힐 경우
- 게임 재시작 기능
- 게임 종료 기능

### 입력

- 3자리 수 입력
- 게임 종료 후 재시작, 종료

### 출력

- 스트라이크, 볼 카운트
- 하나도 없는 경우
- 모두 맞힐 경우

### 예외 사항

- 숫자가 아닐 경우
- 3자리 숫자가 아닌 경우
- 0이 포함되어 있을 경우
- 중복되는 숫자를 입력할 경우

### 커밋 메시지 규칙

```
- 기능(feat): 새로운 기능을 추가
- 버그(fix): 버그 수정
- 리팩토링(refactor): 코드 리팩토링
- 형식(style): 코드 형식, 정렬, 주석 등의 변경
- 테스트(test): 테스트 추가, 테스트 리팩토링
- 문서(docs): 문서 수정
- 기타(chore): 빌드 업무 수정, 패키지 매니저 설정 등 위에 해당되지 않는 모든 변경
```
